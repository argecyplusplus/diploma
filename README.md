# Разработка приложения для моделирования испытаний газотермических покрытий лопаток газотурбинных двигателей

Программный комплекс для расчета процесса формирования и свойств порошковых покрытий лопаток газотурбинных двигателей. Система объединяет геометрическое построение, расчет аэродинамики в FreeFem++ и тепловой анализ на Python.

## Технические требования
* **ОС:** Windows 10/11
* **Python:** 3.11.6
* **FreeFem++:** v4.15 (должен быть установлен по пути `C:\Program Files (x86)\FreeFem++\FreeFem++.exe`)
* **Flask:** 3.0.3

## Структура проекта
Будет дополнена

## Быстрый старт

### 1. Подготовка окружения
Убедитесь, что вы находитесь в корневой директории проекта (команда cd):
```bash
python -m venv venv
.\venv\Scripts\activate
pip install -r requirements.txt
```
### 2. Настройка путей
Если путь к FreeFem++ отличается от стандартного ```C:\Program Files (x86)\FreeFem++\FreeFem++.exe```, необходимо выполнить следующие действия:

1) Добавьте файл .env в корень проекта
2) Пропишите путь к FreeFem++ по ключу __FREE_FEM_PATH__, например
```bash
FREE_FEM_PATH=C:\Program Files (x86)\FreeFem++\FreeFem++.exe
```

### 3. Запуск приложения
```bash
python app.py
```

После запуска откройте браузер и перейдите по адресу: http://127.0.0.1:5000
